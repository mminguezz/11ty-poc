

# Instalar 11ty
npm install -g @11ty/eleventy

# Iniciar un proyecto
```
mkdir 11ty-website-demo && cd 11ty-website-demo
npm init
npm install --save-dev @11ty/eleventy
```

```
"scripts": {
  "start": "npx @11ty/eleventy --serve",
  "build": "npx @11ty/eleventy"
},
```
